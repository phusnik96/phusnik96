GIS Final Project and More

A Simple Mapping Blog by Peter Husnik

Map 1 - School Districts Near Bike Routes

I chose this map as my first for a couple of reasons, lets get into them. The first reason that I chose this map is that it is the first map I created. There is always something special about the first time you do something. The story of this map is a story of urbanism and the car centric nature of American cities. From this map we can see there are very few neighborhoods that have large schools near the bike network. This map taught me how to select features and to how to create new features from the selections. This is a vital GIS skill that I used many times throughout the Semester.

Map 2 - Projections of The World

This is my second map,lets dive into what makes it my second favorite of the semester! First off I enjoy the look of this map, it gives a very retro feeling which I find appealing. The lessons of this map are all about different ways of looking at the Earth. Its important to realize that the map we see in our textbooks and even in our phones. The traditional Robinson projection shows Northern countries to be much larger than they truly are. This projection shows an implicit bias, which I find interesting. I learned an important skill from this project of choosing the correct projection which was useful for the rest of the Semester.

Map 3 - Dueling Views of Chicago

This is my third set of maps, though its my third map I still enjoy it a lot, lets dig into why! I really enjoyed making this set of maps because it allowed me to see a contrasting view of Chicago. It was also a very real world set of maps, looking at both wealth and crime in a detailed way. I learned how to conducted a more real world and per captia analysis of data, which was useful in this Semester and I think will be one of the most useful aspects of this class in the future.
